Circuit: Layout.png

Purpose: An example of using a "Print" class descendant.

The LiquidCrystal library is a "Print" class descendant and
inherits various properties of the Print base class. This example
from the Arduino.cc tutorials shows how to display "Hello World"
on a cheap liquid crystal display.
